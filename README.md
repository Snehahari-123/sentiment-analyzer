# 📊 Sentiment Analyzer using Django

## 🧠 Overview
This is a simple Django web application that performs sentiment analysis on user input text.  
It classifies text into:

- 😊 Positive  
- 😐 Neutral  
- 😞 Negative  

It also stores results in a database and displays analytics using a dashboard.

---

## 🚀 Features

- Sentiment analysis using TextBlob
- Stores all user inputs in database
- History page to view past results
- Dashboard with sentiment statistics
- Pie chart visualization using Chart.js

---

## 🛠️ Tech Stack

- Python
- Django
- HTML, CSS
- SQLite
- TextBlob (NLP)
- Chart.js
